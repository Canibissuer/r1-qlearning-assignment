# rl-qlearning-assignment

This project demonstrates Q-learning using the FrozenLake-v1 environment from Gymnasium.

## What I did
- Explored a Gym environment
- Implemented Q-learning with a Q-table
- Evaluated the learned policy
- Visualized reward progress and the Q-table

## Results
The agent improved over time and learned a policy that achieved a measurable success rate in the FrozenLake environment.

## Reflection
A key challenge was balancing exploration and exploitation. Hyperparameters such as learning rate, discount factor, and epsilon had a strong effect on how quickly and how well the agent learned.
